Thoughts on Progressive Enhancement

Progressive enhancement is a philosophy which tries to provide a base structure of content to as many users as possible. This can be crusial for certain people who want to use a website because it effectivly lays out how you should make your website accessible to everyone. Everyone should have the right to accessing public websites, and progressive enhancement helps to wide the range of accessability.

Progressive enhancement works by starting with a base structure of content and building from there. Firstly, you want to make the base of your code with HTML. Then, you can use css to enhance the experience of the code with designing your website, as well as making it more appealing for others. Lastly, you want to use JavaScript to make sure everything is running correctly and does not break. Having this type of structure is an easy method which can greatly increase accessability.

In conclusion, everyone has the right to accessibility. Wether through dissability or not having the best software to run, certain things can be closed off and unavailable. Being able to build a website using progressive enhancement methods not only makes it easier for the coder, but it also makes it easier for the user.
